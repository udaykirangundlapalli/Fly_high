# ✈️ Fly High | Premium Travel Discovery

Fly High is a modern, responsive web application designed for travelers to explore global destinations, check real-time weather, and curate their own personal travel bucket list.

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

---
🛠️ Built With
HTML5 & CSS3 - Custom layouts and animations.

JavaScript (ES6) - Asynchronous API handling and DOM manipulation.

Bootstrap 4.6 - Responsive grid system.

Particles.js - Interactive hero background.

Wikipedia API - Destination data sourcing.

OpenWeather API - Real-time weather data.

## 🌟 Features

### 🔍 Real-Time Discovery
* **Live Wikipedia Search:** Search for any city or country and get instant data via the Wikipedia API.
* **Weather Insights:** Integrated with OpenWeatherMap API to provide current temperatures and conditions for trending spots.
* **Interactive Gallery:** A curated, masonry-style image gallery with high-quality visual inspiration.

### 🌓 User Experience
* **Dark/Light Mode:** A seamless theme switcher that remembers your preference using `localStorage`.
* **Favorites System:** Save your dream destinations to a personal collection that persists even after closing the browser.
* **Scroll Reveal:** Sections elegantly slide into view as you navigate down the page.
* **Responsive Design:** Fully optimized for desktop, tablet, and mobile devices with a smart "Hamburger" menu.

### 🛠️ Technical Highlights
* **Glassmorphism UI:** Modern frosted-glass navigation bar and UI components.
* **Particle Background:** An interactive particle network in the hero section for a premium feel.
* **Toast Notifications:** Non-intrusive feedback when saving favorites or sending messages.
